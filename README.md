# templed
Calculate and show the room temperature with DHT11 temperature sensor and LEDs on Arduino.

## Requirements

* Arduino UNO
* DHT11 temperature
* 2 x LED (blue and red)
* Jumpers
* 2 x Resistor (220Î©)
* Arduino IDE
